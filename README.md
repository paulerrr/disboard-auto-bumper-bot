# disboard-auto-bumper-bot

based on shroomjaks' project.

---

## Setup

1. Run:
   ```bash
   npm install
   ```

2. Create a `.env` file:
   ```env
   TOKEN=your_discord_user_token_here
   BUMP_CHANNELS=123456789012345678,987654321098765432
   ```

3. Start the bot:
   ```bash
   npm start
   ```
